# Sign Up Form

A sign up page built with HTML and CSS.

🔗 [Live Demo](https://issuereporter1234.github.io/signupform/)

## Features

- Clean, minimal design
- Styled form layout
- Form submits a POST request to [httpbin.org](https://httpbin.org/) for testing

## Tech Stack

- HTML
- CSS

## How to Use

Clone the repo and open `index.html` in your browser — no installation required.

```bash
git clone https://github.com/issuereporter1234/signupform
cd signupform
```
